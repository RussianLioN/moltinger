## ЭКСТРЕННЫЕ СИТУАЦИИ

### "Claude Code не понимает формат ответа"
Скажи:
```
Открой public/app.js, функция mockAdapterTurn (строки 1922-2050). 
Скопируй ТОЧНУЮ структуру ответа и используй её как шаблон в router.js. 
Поля, их имена и вложенность должны совпадать один-к-одному.
```

### "LLM отвечает на английском"
Добавь в system prompt:
```
ОБЯЗАТЕЛЬНО: все ответы, вопросы и тексты — только на русском языке.
```

### "Генерация summary слишком долгая"
Скажи Claude Code:
```
Используй модель claude-haiku-4-5-20251001 вместо sonnet для генерации summary — 
она быстрее и для шаблонной подстановки данных достаточна.
```

### "Фронтенд не обновляется после изменений"
```bash
# Hard refresh в браузере:
Cmd+Shift+R (Mac) / Ctrl+Shift+R (Windows)
```

### "Всё сломалось — откат"
```bash
git stash        # сохранить текущее
git checkout .   # откатить всё к последнему коммиту
npm run dev      # перезапустить
```
