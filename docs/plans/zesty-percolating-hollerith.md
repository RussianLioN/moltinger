# Beads v0.61 Modernization Plan

> **Branch**: `032-beads-v61-modernization`
> **Scope**: Modernize all Beads references from pre-v0.56 patterns to v0.61 API
> **Risk**: Low-Medium (documentation + config changes, no runtime logic changes)
> **Constraint**: Phase 1 must ship atomically — `codex-check.sh` asserts `bd sync` presence
> **Deferred**: `bin/bd`, `beads-resolve-db.sh`, `beads-worktree-localize.sh` rewrite — separate spec due to high risk

---

- Beads v0.61.0 installed (3ac028bf)
