# Полный диалог по OnePage (КПКИ)

## Метаданные
- Сценарий: подготовка `one-page summary` для кредитного комитета.
- Канал: web demo (`demo.ainetic.tech`).
- Источник фактических реплик:
  - пользовательские логи из ручного прогона, переданные в чат;
  - локальные артефакты регрессии (`output/playwright/ui-regression/*`, `output/playwright/arcu4/*`).
- Обозначения:
  - `[LOG]` — фактически зафиксированные реплики;
  - `[GEN]` — логически сгенерированные реплики, добавленные для закрытия сценария до конца.
