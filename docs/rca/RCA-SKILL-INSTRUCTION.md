# Инструкция к навыку RCA (`rca-5-whys`)

## Назначение

Навык `rca-5-whys` нужен для анализа любой ошибки по методу «5 почему», фиксации корневой причины и предотвращения повторения инцидента.

Источник навыка: `.claude/skills/rca-5-whys/SKILL.md`.

## Когда запускать RCA обязательно

- Любой `exit code != 0`
- Ошибка в коде, конфиге, процессе, коммуникации или рассуждении
- Любой незапланированный результат в сессии

## Минимальный протокол (DoD)

RCA считается завершенным только если выполнены все пункты:

1. Создан отчет в `docs/rca/YYYY-MM-DD-topic.md`
2. В отчете заполнен раздел `## Уроки`
3. Выполнен `./scripts/build-lessons-index.sh`
4. Проверена индексация `./scripts/query-lessons.sh --all`
5. Добавлено профилактическое обновление инструкций:
   - `AGENTS.md` или `CLAUDE.md`, либо
   - отдельный файл правила в `docs/rules/*.md`
6. Изменения закоммичены

## Пошаговый workflow

### 1) Зафиксировать ошибку и контекст

Соберите:
- точный текст ошибки
- команду/действие, которое вызвало проблему
- текущую ветку и окружение

### 2) Провести анализ «5 почему»

Формат:
- Почему #1: симптом
- Почему #2: причина уровня 1
- Почему #3: причина уровня 2
- Почему #4: причина уровня 3
- Почему #5: системная корневая причина

### 3) Создать RCA-отчет

Шаблон: `docs/rca/TEMPLATE.md`

```bash
cp docs/rca/TEMPLATE.md "docs/rca/$(date +%F)-<short-topic>.md"
```

### 4) Добавить профилактику в инструкции

Рекомендуемый формат:
- отдельный файл правила в `docs/rules/`
- в `CLAUDE.md`/`AGENTS.md` оставить только короткую ссылку (без раздувания)

### 5) Обновить lessons index

```bash
./scripts/build-lessons-index.sh
./scripts/query-lessons.sh --all
```

### 6) Проверить готовность перед push

Для веток с новыми RCA-файлами должен быть выполнен пункт про обновление инструкций.

```bash
scripts/rca-completion-check.sh --check --range HEAD
```

## Готовый промпт для запуска RCA в новом треде

```text
Примени навык rca-5-whys к ошибке ниже.
Сделай полный протокол 5 почему, сформируй RCA-отчет в docs/rca/,
добавь уроки и предложи/внеси профилактическое обновление инструкций
(AGENTS.md, CLAUDE.md или docs/rules/*.md), затем проверь индекс уроков.

Ошибка: <вставьте текст ошибки>
Контекст: <что запускали, в какой ветке, что ожидали>
```
